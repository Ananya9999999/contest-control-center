# Contest Control Center — CodeChef VIT

Production-style contest administration dashboard built with TanStack Start, React 19, TypeScript and Tailwind CSS v4. All data is mocked locally — no backend required.

## Getting started

```bash
bun install   # or npm install
bun dev       # http://localhost:8080
```

## Scripts

- `dev` – start the dev server
- `build` – production build
- `preview` – preview the production build
- `lint` – run ESLint
